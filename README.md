#### 各部分功能
+ assess 对结果评分并统计
+ data 处理数据集
+ dataset 数据集存放目录
+ nnet 神经网络模型
+ sound 语音特征提取
+ do_assess.py 评估测试输出的表现
+ global_config.py 全局配置，包括日志配置
+ run.py 训练测试运行示例
+ util.py 包含各种辅助处理函数

- Ubuntu16.04 python3.5
